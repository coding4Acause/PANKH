Detailed Description

Provides a function to compute the influence of a linearly varying vortex panel.

This file contains function that compute the influence of a vortex panel of linearly varying strength at an arbitrary point in the flowfield.

Computes the influence matrix for a linearly varying vortex panel.

Given the endpoints of a vortex panel and a desired point in the flowfield, this function calculates the influence matrix, which is useful in solving the panel method for aerodynamic simulations.

Parameters
point1_xX-coordinate of the first endpoint of the panel.
point1_yY-coordinate of the first endpoint of the panel.
point2_xX-coordinate of the second endpoint of the panel.
point2_yY-coordinate of the second endpoint of the panel.
desired_point_xX-coordinate of the target point in the flowfield.(point where the influence needs to be conmputed)
desired_point_yY-coordinate of the target point in the flowfield.(point where the influence needs to be conmputed)
Returns
MatrixXd (2x2) representing the influence matrix corresponding to a particular panel.

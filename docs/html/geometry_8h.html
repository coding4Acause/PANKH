Computes the coordinates of a NACA 4-digit series airfoil at a given chord position.

This function calculates the upper, lower, and camber line coordinates for a NACA 4-digit airfoil based on the non-dimensional chord position and airfoil parameters. Supports both symmetric and cambered airfoils, with open or closed trailing edges.

Parameters
x_cNon-dimensional chord position (x/c), must be in [0, 1].
cChord length (meters).
qNon-dimensional position of maximum camber (fraction of chord, 0 < q < 1).
pMaximum camber (fraction of chord).
trailing_edge_typeTrailing edge type: 1 for open, 0 for closed.
t_mThickness-to-chord ratio (e.g., 0.12 for 12% thickness).
Returns
MatrixXd A 3x2 matrix containing [x_upper, y_upper; x_lower, y_lower; x_camber, y_camber].
Exceptions
std::invalid_argumentIf x_c is outside [0, 1] or other invalid parameters.

Discretizes the airfoil into nodes using cosine clustering.

Generates initial nodal coordinates for a NACA 4-digit airfoil using cosine clustering to concentrate nodes near the leading and trailing edges. Stores results in x0 and y0 vectors and writes them to "output_files/_time=0.dat".

Parameters
nNumber of nodes (even or odd).
cChord length (meters).
qNon-dimensional position of maximum camber (fraction of chord, 0 < q < 1).
pMaximum camber (fraction of chord).
trailing_edge_typeTrailing edge type: 1 for open, 0 for closed.
t_mThickness-to-chord ratio.
x0Output vector for x-coordinates (size n).
y0Output vector for y-coordinates (size n).
Exceptions
std::runtime_errorIf file output fails.
std::invalid_argumentIf n < 2 or other invalid parameters.

Transforms airfoil nodal coordinates to the instantaneous inertial frame.

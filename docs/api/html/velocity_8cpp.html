Computes the velocity induced at a point by bound vortices on an airfoil.

This function calculates the velocity at a given point P(x, y) location due to the influence of bound vortices along the airfoil surface. It uses the influence matrix to determine the contribution of each vortex panel.

Parameters
x_ppVectorXd containing x-coordinates of the panel endpoints.
y_ppVectorXd containing y-coordinates of the panel endpoints.
xDouble value representing the x-coordinate of the evaluation point.
yDouble value representing the y-coordinate of the evaluation point.
G_boundVectorXd containing the circulation strengths of the bound vortices.
Returns
VectorXd A 2D velocity vector [Vx, Vy] induced at the given (x, y) location.

Computes the velocity induced at a point by a single discrete vortex.

This function calculates the velocity induced at a given (des_point_x, des_point_y) location due to a discrete vortex of strength gamma located at (vor_point_x, vor_point_y). It follows the Biot-Savart law to determine the velocity components. This function will be required to compute the velocity induced due to shed vortices in the wake.

Parameters
gammaDouble value representing the circulation strength of the vortex.
vor_point_xDouble value representing the x-coordinate of the vortex.
vor_point_yDouble value representing the y-coordinate of the vortex.
des_point_xDouble value representing the x-coordinate of the evaluation point.
des_point_yDouble value representing the y-coordinate of the evaluation point.
Returns
VectorXd A 2D velocity vector [Vx, Vy] induced at the given (des_point_x, des_point_y) location.
